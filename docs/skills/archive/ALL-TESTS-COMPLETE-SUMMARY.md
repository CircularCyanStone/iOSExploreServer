## 测试挑战与解决方案

### 挑战 1: ui.navigation.tapBarButton 参数理解
**问题**: 最初误认为 `title` 可以单独用于选择按钮  
**解决**: 查看源代码确认 `title` 仅用于验证  
**教训**: 在测试前先理解命令的设计意图

### 挑战 2: ui.scrollToElement 元素定位
**问题**: 最初按 UICollectionViewCell 类型查找，找不到元素  
**解决**: 改为按 text 内容查找  
**教训**: UI 层级可能与预期不同，需要灵活调整

### 挑战 3: 导航和状态管理
**问题**: 测试过程中需要在多个页面间导航  
**解决**: 使用 `ui.navigation.back` 返回主菜单，重新导航  
**教训**: 测试脚本需要做好状态管理
