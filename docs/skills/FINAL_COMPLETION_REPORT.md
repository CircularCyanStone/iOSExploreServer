## 项目价值

### 对开发者
- 从"需要记住一堆命令"变为"一句话说明需求"
- 从"看英文错误日志"变为"直接获得中文操作指引"

### 对测试人员
- 不需要理解 USB UDID vs CoreDevice ID
- 不需要手动诊断端口冲突
